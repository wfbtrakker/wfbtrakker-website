  • Dell, HP, LG, Samsung, ASUS, Acer, BenQ, ViewSonic
  • Any monitor supporting DDC/CI
  • HDMI, DisplayPort, and DVI connections

Built on Modern Technology

  • Native .NET 8.0 for reliability
  • Direct Windows API integration
  • Lightweight with no unnecessary dependencies

System Requirements

  • .NET 8.0 Desktop Runtime

  • Less than 5 MB disk space
